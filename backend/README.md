# NirvachakSetu — Backend

This is the Node.js/Express backend for the NirvachakSetu Election Education Platform.

## 🚀 Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Configure environment variables:
   Create a `.env` file with the following:
   ```env
   PORT=5000
   MONGODB_URI=your_mongodb_connection_string
   GEMINI_API_KEY=your_gemini_api_key
   ```

3. Run the server:
   ```bash
   node server.js
   ```
