# NirvachakSetu — Frontend

This is the React-based frontend for the NirvachakSetu Election Education Platform.

## 🚀 Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```
